# @kyvrixon/utils

General utility files I use for alot of projects! Does not include any larger scale items, just the little things. Designed for use with [bun runtime](https://bun.sh/)!

```bash
bun install @kyvrixon/utils
# or
bun install github:kyvrixon/utils#1.0.0
# if you want the absolute latest changes (unstable)
bun install github:kyvrixon/utils#main
```
